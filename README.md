# pc-uedsport-lab

一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何特定网站或域名，仅作为静态页面的存放与展示用途。

## 项目简介

本仓库主要面向实验性、存档性的 HTML 页面收集与分发。页面内容涵盖不同主题和用途，彼此独立，不构成完整网站。适合用于技术演示、学习参考或临时页面托管。

## 目录说明

```
├── index.html          # 仓库入口页（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：所有独立 HTML 页面存放于此，每个文件可独立访问。
- `assets/`：页面共享的静态资源，按类型分目录存放。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖其他页面。
- 页面可引用 `assets/` 下的公共资源，但建议尽量内嵌或自包含。
- 本仓库不保证页面内容长期可用或更新，仅供存档参考。
- 页面内容可能涉及不同来源、不同时期的资料，请自行鉴别。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 以修正错误或添加新页面。
- 新增页面请放入 `pages/` 目录，并尽量保持资源文件整洁。
